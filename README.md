# time-reminder
Show a simple TOPMOST reminder elapsing or timeout timer for meeting

## masm64 subdirectory ##

MASM x64 implement Reminder

```shell
reminder [minutes]
- minutes:   minutes to be tick, then tick will change to 'timeout',default 1 minute
```

## dotnet subdirectory ##

dotnet 7 implement Reminder

```shell
reminder [minutes]
- minutes:   minutes to be tick, then tick will change to 'timeout',default 1 minute
```
